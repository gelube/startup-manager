# 启动管理器 Pro v3.0

一款现代化的 Windows 启动项管理工具，支持注册表、启动文件夹、计划任务和 Windows 服务的管理。

## 功能特性

- **多类别管理**：注册表、启动文件夹、计划任务、Windows 服务
- **批量操作**：支持多选批量启用、禁用、删除
- **状态同步**：实时显示启动项状态（已启用/已禁用/已运行/已停止）
- **现代界面**：简约风格，圆角卡片设计
- **右键菜单**：快速访问常用操作

## 系统要求

- Windows 10/11
- .NET 6.0 Runtime

## 编译

```bash
cd StartupManagerPro
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
```

输出位置：`bin\Release\net6.0-windows\win-x64\publish\StartupManagerPro.exe`

## 使用说明

### 界面布局

- **左侧分类**：按类型筛选启动项
- **右侧列表**：显示启动项详情
- **工具栏**：添加、启用、禁用、删除操作

### 操作方式

- **左键单击**：选中并勾选单行
- **点击复选框**：多选
- **表头复选框**：全选/取消全选
- **右键菜单**：更多操作选项

### 右键菜单

| 类别 | 启用/禁用 | 启动/停止 | 其他选项 |
|------|-----------|-----------|----------|
| 注册表 | 隐藏 | 隐藏 | 打开位置、属性、删除 |
| 启动文件夹 | 显示 | 隐藏 | 打开位置、属性、删除 |
| 计划任务 | 显示 | 隐藏 | 打开位置、属性、删除 |
| 服务 | 显示 | 显示 | 打开位置、属性、删除 |
